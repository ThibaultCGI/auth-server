# Auth Server

Projet personnel visant à construire un serveur d'authentification OAuth2/OpenID Connect moderne avec :

- Java 25
- Spring Boot 4
- Spring Security
- Spring Authorization Server
- PostgreSQL
- Liquibase
- Architecture hexagonale

## Objectifs

- Comprendre OAuth2
- Comprendre OpenID Connect
- Comprendre Spring Security
- Mettre en œuvre une architecture hexagonale
- Approfondir Java 25
